SLA Step 1: Create the Network 1. Open GNS3 and drag two Cisco 3725 routers onto the workspace. 2. Connect the two routers using their FastEthernet0/0 interfaces. 3. Assign IP addresses to the interfaces. R1 ---------- R2 On Router 1 (R1): interface FastEthernet0/0 ip address 192.168.1.1 255.255.255.0 no shutdown On Router 2 (R2): interface FastEthernet0/0 ip address 192.168.1.2 255.255.255.0 no shutdown 4. Test connectivity using a simple ping: ping 192.168.1.2 ________________________________________ Step 2: Configure IP SLA on Router 1 1. Enable IP SLA on R1 to ping R2 every 5 seconds: ip sla 1 icmp-echo 192.168.1.2 frequency 5 exit 2. Start the IP SLA operation: ip sla schedule 1 start-time now life forever ________________________________________ Step 3: Check IP SLA Performance â€¢ View the statistics on R1: show ip sla statistics This will show: o Round Trip Time (RTT) o Success or failure of pings ________________________________________ Step 4: Test the Configuration 1. To simulate failure, shut down R2's interface: interface FastEthernet0/0 shutdown 2. Check the IP SLA statistics on R1 again: show ip sla statistics
